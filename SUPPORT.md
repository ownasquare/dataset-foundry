# Support

## Questions and setup help

Use [GitHub Discussions](https://github.com/ownasquare/dataset-foundry/discussions). Include your
operating system, Python version, install path (Docker or source), and the output of
`dataset-foundry doctor` with all secret values removed.

## Reproducible bugs

Open a bug report with the provided issue form. Include the smallest safe reproduction and whether
the problem affects offline, live-provider, browser, package, or container use. Never attach API
keys, customer seeds, private prompts, generated customer data, databases, or exports.

## Security issues

Do not use Discussions or public issues. Follow [SECURITY.md](SECURITY.md).

## Scope

Community support covers the current `main` branch and, after releases begin, the latest tagged
release. Hosted deployment, multi-host queues, production identity, and object-storage integrations
are environment-specific and may require separate operational work.
